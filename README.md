# 小树暑托

一个面向孩子暑假托管场景的手机端单页原型，围绕「计划 → 打卡 → 统计 → 积分奖励」建立闭环。

## 打开方式

直接打开 `index.html` 即可使用，也可以用任意静态文件服务器预览。

管理员后台入口：`admin.html`。首次启用时，先在 App 注册普通邮箱账户，再在后台登录并输入一次性管理员绑定码；成功领取后，App 的「我的」页面会显示管理员后台入口。

## 当前功能

- 今日计划：学习、运动、生活任务和完成进度
- 计划管理：按今天、明天、后天新增、编辑、删除计划
- 一键打卡：单项或整天完成，积分和统计实时联动
- 成长统计：连续打卡、本周柱状统计、学习洞察
- 积分奖励：自定义奖励、余额、兑换和积分收支记录
- 奖励领取闭环：孩子兑换后生成待领取记录，管理员可确认已经发放
- 通知中心：集中提示今天未完成计划和等待领取的奖励
- 云端保存：Supabase Auth 登录，计划、打卡与积分跨设备同步
- 手机安装：支持添加到主屏幕，并缓存核心页面供离线打开
- 用户账户：支持邮箱注册、邮箱验证、真实登录、退出登录和多用户数据隔离
- 管理员后台：支持一次性安全绑定、用户搜索、启停账户、资料与积分编辑、数据重置和用户删除
- 账户工具：支持资料与密码修改、周报分享、个人数据备份恢复和管理员批量导出

### 免费套餐护栏

- Supabase 组织固定使用 Free Plan，并保持 Spend Cap 开启
- 不绑定付款方式，不自动升级套餐，不购买 Compute、Credit 或其他附加项
- 当前应用仅使用 Auth、Database 和 RPC，不使用 Storage、Realtime、Edge Functions、SSO 或图片转换
- `free-plan-guard.sh` 会在提交和自动同步前扫描代码；如果误接高用量功能或前端 Secret Key，将阻止提交
- 免费额度耗尽时允许服务暂停或进入只读状态，不允许产生超额费用

手动检查护栏：

```bash
./free-plan-guard.sh
```
